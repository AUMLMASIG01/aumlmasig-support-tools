# AUMLMASIG 擴充卡一鍵診斷器人工視覺驗收清單

## 驗收範圍

- 本清單只適用於 `aumlmasig-expansion-card-diagnoser/`。
- 本階段只做人工作業檢查與上線前風險控管。
- 不登入 Shopline、不發布網站、不新增外部套件、不新增後端或資料庫。

## 桌機版檢查項目

- [ ] 使用桌機瀏覽器開啟 `index.html`，頁面標題與主標題清楚顯示。
- [ ] 首屏可理解這是「擴充卡一鍵診斷器」與三步驟選型工具。
- [ ] Step 1、Step 2、Step 3 排版清楚，沒有文字重疊、截斷或不自然換行。
- [ ] 文字輸入、單選、大按鈕選項、多選 chip、textarea 都能操作。
- [ ] 主要按鈕高度足夠，滑鼠點擊目標明確。
- [ ] 診斷後結果區能清楚顯示風險等級、建議卡種、方案、提醒、確認事項與診斷單。
- [ ] 清除重填後表單與結果區回到初始狀態。
- [ ] 畫面沒有橫向捲動或元件超出視窗。

## 手機版 390px 檢查項目

- [ ] 用瀏覽器開發者工具切到 390px 寬度。
- [ ] 主標題、副標、三個步驟與表單欄位在手機版可讀。
- [ ] 所有選項可用手指點選，不需要鍵盤多選操作。
- [ ] 主要按鈕高度至少 44px，按鈕文字沒有擠壓。
- [ ] 結果區在手機版垂直排列合理，風險標籤明顯。
- [ ] 診斷單 textarea 可閱讀、可選取、可複製。
- [ ] 商品按鈕、診斷單複製按鈕不互相重疊。
- [ ] 沒有橫向捲動、文字遮擋或內容被切掉。

## 平板版 768px 檢查項目

- [ ] 用瀏覽器開發者工具切到 768px 寬度。
- [ ] 表單欄位排列自然，兩欄與單欄轉換不造成閱讀順序混亂。
- [ ] chip 與 radio 卡片大小一致，點擊區域明確。
- [ ] 結果區資訊層級清楚，不像塞滿的表格。
- [ ] 可複製診斷單區塊有足夠高度，按鈕列不換到難以理解的位置。

## Shopline 嵌入前檢查項目

- [ ] 開啟 `shopline-embed.html`，確認內容可直接複製。
- [ ] 貼上前確認 CSS class 都使用 `auml-` 前綴。
- [ ] 貼上前確認樣式只作用於診斷器容器與其內部元素。
- [ ] 貼上前確認檔案不依賴本機相對路徑、外部建置工具或後端服務。
- [ ] 未設定下載中心與客服連結時，不顯示對應按鈕或壞連結。
- [ ] 高風險結果不顯示推薦商品按鈕。
- [ ] 高風險結果顯示客服協助確認表入口。
- [ ] Shopline 預覽中需再次檢查桌機、手機、平板三種寬度。

## 結果區檢查項目

- [ ] 低風險顯示「低風險：可直接購買」。
- [ ] 中風險顯示「中風險：建議先確認」。
- [ ] 高風險顯示「高風險：請先聯繫客服，不建議直接下單」。
- [ ] 診斷摘要有建議卡種。
- [ ] 建議方案固定顯示超值方案、穩定方案、專業方案。
- [ ] 相容提醒使用保守語氣，不做保證相容或一定可用承諾。
- [ ] 購買前確認事項能引導客戶提供主機板、CPU、作業系統與用途。
- [ ] 商品按鈕只在有正式分類連結且不是高風險時顯示。

## 診斷單複製檢查項目

- [ ] 完成診斷後，診斷單包含「AUMLMASIG 擴充卡一鍵診斷單」。
- [ ] 診斷單包含診斷時間、使用裝置、機器類型、主要需求、痛點、PCIe 插槽、M.2 類型、長時間使用、作業系統、補充說明。
- [ ] 診斷單包含風險等級、系統建議、相容提醒、購買前確認事項。
- [ ] 點擊「一鍵複製診斷單」後顯示成功訊息。
- [ ] 貼到文字編輯器或客服對話框草稿時，內容完整且換行正常。

## 高風險不導購檢查項目

- [ ] 選擇「沒有 / 筆電或迷你主機無法裝」時，風險為高風險。
- [ ] 筆電且 PCIe 不確定時，結果提醒筆電通常不能安裝標準 PCIe 擴充卡。
- [ ] 高風險結果不顯示「查看推薦商品」按鈕。
- [ ] 高風險結果顯示「請先聯繫客服確認」與客服協助確認表入口。
- [ ] 診斷單仍可複製，方便客戶交給客服確認。

## CEO 人工測試案例

| 案例 | 輸入摘要 | 預期結果 | 驗收 |
| --- | --- | --- | --- |
| 1 | 桌機 + 增加 NVMe M.2 SSD + PCIe x16 + Windows 11 | 低風險，建議 NVMe 擴充卡，可顯示推薦商品 | [ ] |
| 2 | 筆電 + 增加 NVMe + PCIe 不確定 | 高風險或中高風險，提醒筆電通常無法安裝標準 PCIe 擴充卡 | [ ] |
| 3 | 多顆 NVMe / RAID + PCIe x1 | 中風險以上，提醒 x1 頻寬不足與 Bifurcation | [ ] |
| 4 | 增加 SATA 硬碟 + 長時間監控 | 建議 SATA 擴充卡，提醒穩定性與散熱 | [ ] |
| 5 | 增加 USB 埠 + 補充說明 20Gbps | 建議 USB 擴充卡，提醒 PCIe 頻寬與裝置支援 | [ ] |
| 6 | 增加 10G 網路 | 建議網路卡，提醒 PCIe 頻寬、交換器、網路線、散熱 | [ ] |
| 7 | 沒有 PCIe 插槽 | 高風險，不顯示直接購買導向，建議聯繫客服 | [ ] |
