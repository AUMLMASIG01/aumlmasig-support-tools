# AUMLMASIG 擴充卡一鍵診斷器手動測試案例

## 測試前檢查

- 使用本機瀏覽器開啟 `index.html`。
- 瀏覽器寬度調整到 390px，確認選項、按鈕、結果與診斷單可正常操作。
- 開啟瀏覽器 console，確認操作期間沒有錯誤訊息。
- 未設定下載中心與客服連結時，不應顯示對應按鈕。
- 商品連結只應依 `src/config.js` 中已確認的分類連結顯示。

## 案例 1：桌機 + 增加 NVMe M.2 SSD + PCIe x16 + Windows 11

輸入：

- 機器類型：桌上型電腦
- 主要需求：增加 NVMe M.2 SSD
- PCIe 插槽：有 PCIe x16
- M.2 類型：NVMe
- 長時間使用：不會，一般使用
- 作業系統：Windows 11

預期：

- 風險等級：低風險
- 建議卡種：NVMe M.2 擴充卡
- 顯示 NVMe 分類商品連結
- 可產生完整診斷單

## 案例 2：筆電 + 增加 NVMe + PCIe 不確定

輸入：

- 機器類型：筆電
- 主要需求：增加 NVMe M.2 SSD
- PCIe 插槽：不確定
- M.2 類型：不確定

預期：

- 風險等級：高風險
- 相容提醒包含筆電通常不能安裝標準 PCIe 擴充卡
- 診斷單可複製給客服確認

## 案例 3：多顆 NVMe / RAID + PCIe x1

輸入：

- 機器類型：桌上型電腦
- 主要需求：多顆 NVMe / RAID / 高速工作流
- PCIe 插槽：有 PCIe x1
- 痛點：速度不穩 / 會掉速

預期：

- 風險等級：中風險以上
- 相容提醒包含 PCIe x1 頻寬不足
- 相容提醒包含 PCIe Bifurcation

## 案例 4：增加 SATA 硬碟 + 長時間監控

輸入：

- 機器類型：NAS / 監控主機
- 主要需求：增加 SATA 硬碟
- PCIe 插槽：有 PCIe x4
- 長時間使用：會，渲染 / 備份 / 伺服器 / 監控 / 長時間運作

預期：

- 建議卡種：SATA 擴充卡
- 相容提醒包含穩定性與散熱
- 三種方案都正常顯示

## 案例 5：增加 USB 埠 + 補充說明 20Gbps

輸入：

- 主要需求：增加 USB 埠
- PCIe 插槽：有 PCIe x4
- 補充說明：需要 USB 20Gbps

預期：

- 建議卡種：USB 擴充卡
- 相容提醒包含主機 PCIe 頻寬與前端裝置支援
- 顯示 USB 分類商品連結

## 案例 6：增加 10G 網路

輸入：

- 主要需求：增加網路埠
- PCIe 插槽：有 PCIe x4
- 補充說明：需要 10G 網路

預期：

- 建議卡種：網路擴充卡
- 相容提醒包含 PCIe 頻寬、交換器、網路線、散熱
- 顯示網路卡分類商品連結

## 案例 7：沒有 PCIe 插槽

輸入：

- 主要需求：增加 NVMe M.2 SSD
- PCIe 插槽：沒有 / 筆電或迷你主機無法裝

預期：

- 風險等級：高風險
- 結果標題：目前不建議直接購買 PCIe 擴充卡
- 相容提醒包含無 PCIe 插槽設備通常無法安裝 PCIe 擴充卡

## 複製功能檢查

- 任一案例完成診斷後點擊「一鍵複製診斷單」。
- 預期顯示「已複製，可貼給客服確認」。
- 貼到文字編輯器後應包含診斷時間、使用裝置、主要需求、風險等級、相容提醒與購買前確認事項。
